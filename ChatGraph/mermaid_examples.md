# Mermaid关系图示例

本文档提供了各种企业知识图谱关系图的Mermaid代码示例，供大模型参考生成。

## 1. 投资关系图

### 个人投资公司
```mermaid
graph LR
    张伟 -->|投资| 阿里巴巴集团
    张伟 -->|投资| 腾讯控股有限公司
    张伟 -->|投资| 百度公司
```

### 公司投资关系链
```mermaid
graph LR
    阿里巴巴集团 -->|投资| 蚂蚁金服
    阿里巴巴集团 -->|投资| 菜鸟网络
    腾讯控股有限公司 -->|投资| 微信支付
    腾讯控股有限公司 -->|投资| 腾讯云
```

### 多层投资关系
```mermaid

    张伟 -->|投资| 阿里巴巴集团
    阿里巴巴集团 -->|投资| 蚂蚁金服
    蚂蚁金服 -->|投资| 支付宝
```

## 使用说明

1. **实体命名**：使用查询结果中的实际实体名称
2. **关系描述**：使用中文描述关系类型（投资、分支机构、任职等）
3. **图形方向**：使用 `graph LR`（从左到右）或 `graph TD`（从上到下）
4. **样式**：保持简洁，避免过于复杂的图形
5. **数量控制**：当关系过多时，选择最重要的关系进行展示

## 注意事项

- 只在实际存在关系时才生成图形
- 实体名称要准确，避免歧义
- 关系类型要清晰明确
- 图形布局要合理，避免交叉
- 当关系网络过于复杂时，可以分多个图形展示
